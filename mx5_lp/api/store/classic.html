The "Classic" icon theme pulled out from Samsung 4.4.4 builds.

This is a theme created using the Miix ThemePack API (MTPAPI) platform, designed to fully theme devices with Miix (a ROM) installed, and also to support a wide array of launchers, not limited to:

- ADW.Launcher

- Nova Launcher

- Apex Launcher

DOWNLOAD THEME

HOW TO INSTALL?

Install the theme


Apply on
Settings > Wallpapers and themes > Themes

Reboot and enjoy!
